:focus-visible 선택자 예제

키보드로 Tab 키를 눌러 포커스를 이동하면 포커스 링이 보이지만, 마우스로 클릭하면 보이지 않습니다.

예시 1: 버튼의 키보드 포커스만 표시

Tab 키로 이동하면 파란색 포커스 링이 보입니다. 마우스로 클릭하면 보이지 않습니다.

예시 2: 링크의 키보드 포커스만 강조

링크 1 링크 2 링크 3

Tab 키로 이동하면 노란 배경과 파란 테두리가 나타납니다.

예시 3: 입력 필드의 키보드 포커스만 표시


Tab 키로 이동하면 파란 테두리와 그림자가 나타납니다.

참고: 이 페이지에서 Tab 키를 눌러 포커스를 이동해보세요. 키보드로 탐색할 때만 포커스 표시가 나타나는 것을 확인할 수 있습니다.
