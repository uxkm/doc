:focus-within 선택자 예제

자식 요소에 포커스가 있으면 부모 요소의 스타일이 변경됩니다.

예시 1: 폼 필드에 포커스가 있을 때 폼 박스 스타일 변경

입력 필드에 포커스를 주면 부모 폼 박스의 테두리와 배경색이 변경됩니다.

예시 2: 검색창 내부에 포커스가 있을 때 검색창 강조

검색 입력 필드나 버튼에 포커스를 주면 검색창 전체에 그림자가 생깁니다.

예시 3: 드롭다운 메뉴 내부에 포커스가 있을 때 메뉴 활성화

메뉴 항목 1 메뉴 항목 2 메뉴 항목 3

드롭다운 버튼이나 메뉴 항목에 포커스를 주면 메뉴가 표시됩니다.
