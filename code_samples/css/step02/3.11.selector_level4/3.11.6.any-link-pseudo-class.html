:any-link 선택자 예제

방문한 링크든 방문하지 않은 링크든 상관없이 모든 링크에 같은 스타일이 적용됩니다.

예시 1: 모든 링크에 공통 스타일 적용

방문하지 않은 링크 1 방문하지 않은 링크 2 외부 링크 (방문 후 보라색이 되지 않음)

모든 링크가 파란색으로 표시되며, 방문 여부와 관계없이 동일한 스타일이 적용됩니다.

예시 2: 링크에 아이콘 추가

페이지 1 페이지 2 페이지 3

모든 링크 뒤에 화살표 아이콘이 자동으로 추가됩니다.

예시 3: 네비게이션 링크 스타일 통일

홈 소개 서비스 연락처

네비게이션의 모든 링크에 동일한 패딩, 테두리, 호버 효과가 적용됩니다. 방문 여부와 관계없이 일관된 스타일을 유지합니다.
